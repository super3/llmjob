How to Add a Node

1. Install the LLMJob Node Client

npm install -g llmjob-node

2. Start Your Node

llmjob-node start --name "My Node"

3. Claim Your Node

Visit the URL shown in the client output to claim the node.

Quick Start

# Install OpenAI SDK
pip install openai
# Python example
from openai import OpenAI
client = OpenAI(
api_key="lj-...",
base_url="https://llmjob.com/api/v1"
)
response = client.chat.completions.create(
model="llama2-7b",
messages=[{"role": "user", "content": "Hello!"}]
)
